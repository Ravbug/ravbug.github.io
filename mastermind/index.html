How to Play:

I have devised a secret code of 4 colors. Try to guess it! The code can contain repeated colors.

Click the round pegs on the left column of the board to set their color. Press Check if you think you have the correct answer.


If you have a pin in the correct position with the correct color, you will see a red pin on the right side of the board, as shown above.

If you have a pin with the correct color but in an incorrect position, you will see a white pin as shown above.

The placement of my hint pins does not correspond to the placement of the pins in your guess.

If you determine my secret code within 10 attempts, you win!
