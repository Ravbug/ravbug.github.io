Welcome to the PyPi Drinking Game! "Python enables you to write code faster" ... if you make extensive use of third party libraries. Because code written by strangers on the internet is guaranteed to be safe, secure, and performant, right?

The rules of the game are very simple.

  1. Think of a trivial programming task. The more trivial, the better.
  2. Think of a likely name for a PyPi module that solves that task.
  3. If a module exists to solve that problem, check the following criteria:
    1. If the module has less than 10kb of code, excluding dependencies, take 1 shot.
    2. if the module is small and has 3 or more dependencies, take 1 more shot
    3. If the module has dependencies, take 1 shot for every additional module, for each of the above criteria it satisfies.
  4. Tally up your score and drink up! (please don't actually drink)
