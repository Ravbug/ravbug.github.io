Extracting Minecraft Texture Files

From any version using its jar file

I cannot release a download to these images because that would violate the Minecraft license. However, I can tell you how to get them off your own computer. To get the files, do the following:

All platforms:
Legacy Java Launcher
  1. Start the minecraft launcher
  2. Click the Edit Profile button
  3. Click the Open Game Dir button (A Finder of File Explorer window will pop up)
  4. Inside the Finder/File Explorer Window, go to the versions folder
  5. Find the version you want and open the folder*
  6. Follow your platform-specific instructions below
New (chromium) Launcher
  1. Start the Minecraft launcher
  2. Open the hamburger menu at the top right, and open Launch Options
  3. Click on Latest release
  4. Next to the row that says "Game Directory", click the little green arrow
  5. Inside the Finder / File Explorer window, go to the versions folder
  6. Find the version you want and open the folder*
  7. Follow your platform-specific instructions below
Mac
  1. Right-click the .jar folder and choose Open With -> Archive Utility
  2. Find the assets folder and go to assets/minecraft/textures
  3. Enjoy the texture files!
Windows/Linux
  1. Make a duplicate of the .jar file
  2. Change the duplicate's extension to .zip
  3. Open the .zip file
  4. Find the assets folder and go to assets\minecraft\textures
  5. Enjoy the texture files!
