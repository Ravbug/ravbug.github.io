"How package is this??"

After (almost) every major Minecraft Java Edition Update, starting with version 1.14, I run every single string in the game through multiple recursive layers of Google Translate. Yes, every single one. Even the menus, splashes, and end poem. Some example screenshots are below. The customizer allows you to choose which translation variant for each Minecraft Version you want to include in the pack that you download.

How to install these packs (read carefully!)

  1. Place the .zip files inside your resourcepacks folder at minecraft/resourcepacks
  2. Open Minecraft, select Options > Resource Packs, and enable the resource pack you want to use. The language mods will not be active right away.
  3. Important! Return to the Options menu, select Language, then find Google Translate in the list and select it.
  4. Return to the menu. The text should be changed.
