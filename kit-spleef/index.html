Important Notes

  • If players are unable to change settings, set spawn_permission in server.properties to 0.
  • This map will probably not work on servers with plugins! Please use vanilla without datapacks to ensure the best results. I will not fix issues relating to plugin compatibility. Instead, report those to the plugin devs.
