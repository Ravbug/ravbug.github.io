Troubleshooting
  • Problem: Lobby does not load / players spawned outside the lobby
    Solution: /function tntwars:lobby
  • Problem: Players cannot edit settings via clicking the signs
    Solution: edit server.properties, set spawn-protection=0
  • Problem: game doesn't start or datapack doesn't initialize
    Solution: /reload with at least 1 player in the world
  • Problem: round doesn't end
    Solution: /kill @e[type=villager], and if that doesn't work, /function tntwars:gameover
  • Problem: late-joiner spawns in the wrong location
    Solution: /execute as PLAYERNAME run function tntwars:join_blue / /execute as PLAYERNAME run function tntwars:join_red, then /kill PLAYERNAME
  • Problem: none of the above steps worked
    Solution: /stop, delete all folders inside world except datapacks, and restart the server
