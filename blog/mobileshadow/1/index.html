MobileShadow is a research project investigating rendering pixel-accurate raster-shadows in real-time using mobile-friendly techniques.

    Below is a list containing relevant prior research
  • An Efficient Alias-free Shadow Algorithm for Opaque and Transparent Objects using per-triangle Shadow Volumes

    Via a custom CUDA-based layered rasterizer, generate shadow volumes per triangle, enabling support for arbitrary geometry and transparency

  • Alias-Free Shadow Maps

    By changing the way a shadow map is arranged, produce pixel-perfect shadows using depth mapping.

  • Pixel Accurate Shadows with Shadow Mapping

    Combining conventional depth mapping with a multipass confidence-based sampling system to produce pixel-perfect shadows with shadow mapping

  • Shadow Volume Extrusion using a Vertex Shader

    Preprocessing geometry to add zero-length quads, such that volume extrusion can be performed in a vertex shader.

  • Rendering 3D Vector Data Using the Theory Of Shadow Volumes

    An overview of the shadow volumes technique.

  • Emulation of Geometry Shaders and Stream Output Using Compute Shaders

    A patent describing a method of emulating the featurset and behavior of geometry shaders using compute shaders. Will need to ensure my implementation, however it ends up working, is not infringing it.

  • Efficient Ray Tracing of Large 3D Scenes for Mobile Distributed Computing Environments

    Implementing ray tracing in a manner suitable for render farms composed of mobile phone hardware

  • Method for Rendering Shadows Using a Shadow Volume and a Stencil Buffer

    The now-expired 1999 patent describing the z-fail shadow volume algorithm, which is informally known as Carmack's Reverse.
